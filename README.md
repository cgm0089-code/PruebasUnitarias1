  # Proyecto de Pruebas Unitarias

  Este proyecto contiene una clase `ValidadorRegistroTest` que realiza pruebas unitarias para validar el comportamiento de la clase `ValidadorRegistro`.

  ## Pruebas Implementadas

  1. **Ejemplo: Un nombre válido debe ser aceptado**
     - Verifica que un nombre válido, como "Carmen", sea aceptado por el validador.

  2. **Reto 1: Un nombre vacío debe ser rechazado**
     - Verifica que un nombre vacío `""` sea rechazado por el validador.

  3. **Reto 2: Una contraseña de 8 caracteres debe ser aceptada**
     - Verifica que una contraseña válida de 8 caracteres sea aceptada por el validador.

  4. **Reto 3: Una contraseña de menos de 8 caracteres debe ser rechazada**
     - Verifica que una contraseña de menos de 8 caracteres sea rechazada por el validador.

  5. **Reto 4: Un email sin arroba debe ser rechazado**
     - Verifica que un email sin arroba `"usuario@"` sea rechazado por el validador.

  6. **Reto 5: Un usuario de más de 18 caracteres debe ser rechazado**
     - Verifica que un usuario de más de 18 caracteres sea rechazado por el validador.

  ## Ejecución de Pruebas

  Para ejecutar las pruebas unitarias, sigue estos pasos:

  1. Abre el proyecto en tu IDE preferido.
  2. Localiza la clase `ValidadorRegistroTest`.
  3. Ejecuta la clase como una prueba unitaria.
  4. Verifica que todas las pruebas pasen exitosamente.

  ¡Disfruta del desarrollo de este proyecto de pruebas unitarias!
